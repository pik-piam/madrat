# madTempDir

returns a temporary directory as a subfolder of the tempfolder set in
`getConfig("tmpfolder")`.

## Usage

``` r
madTempDir()
```

## Value

path to the temp folder

## See also

[`getConfig`](getConfig.md)

## Author

Jan Philipp Dietrich

## Examples

``` r
if (FALSE) { # \dontrun{
madrat:::madTempDir()
} # }
```
