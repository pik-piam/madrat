# Convert Tau

Convert landuse intensity data (tau) to data on ISO country level.

## Usage

``` r
convertTau(x)
```

## Arguments

- x:

  MAgPIE object containing tau values and corresponding weights xref at
  0.5deg cellular level.

## Value

Tau data and weights as MAgPIE object aggregated to country level
